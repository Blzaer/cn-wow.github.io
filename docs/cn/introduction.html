# 服务器特色

  • 这是个长久耐玩服，不收费，2014年开服，100%原汁原味服，人气4000-9000，非练习服也不是魔改服，不适合快餐党和速成党。
  • 一倍练级经验&专业经验，非DK职业练满级大约需要120-200小时游戏时间，野怪视野增大，icc副本0buff且boss血量增加、技能伤害增加、狂暴时间缩短。
  • 没有G团，没有外挂，也没有工作室。
  • 公会欢迎所有小团队和散人入驻！
